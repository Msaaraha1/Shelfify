## 📌 About The Project

Shelflify is a beginner-friendly frontend project developed to practice real-world JavaScript concepts such as:

* DOM Manipulation
* Event Handling
* Dynamic UI Rendering
* Form Input Handling
* Interactive Web Design

It allows users to add, display, and remove books dynamically with a professional user experience.

---

## ✨ Features

✅ Add books with title, author, and description
✅ Display books in responsive cards
✅ Delete books instantly
✅ Floating add button
✅ Popup modal form
✅ Clean modern UI
✅ Fully responsive layout
✅ Beginner-friendly code structure
